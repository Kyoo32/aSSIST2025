# aSSIST2025

## About This Program
이 저장소는 **aSSIST 서울과학종합대학원대학교(aSSIST University) AI·빅데이터 석사 과정** 학습 기록입니다.  
2025-2학기 전공기초 기준으로, AI와 데이터 사이언스의 핵심 영역을 모듈형으로 학습하며 이론과 실습을 함께 진행합니다.

- 과정: AI·빅데이터 석사
- 학교: aSSIST 서울과학종합대학원대학교 (aSSIST University)
- 학기 운영(전공기초 기준): 금요일/토요일 집중 수업
- 모듈 구성 예시:
- Python
- Mathematics with Python
- Machine Learning
- Deep Learning
- AI Vision
- 개인화추천
- AI 시대의 프로그래밍
- AI연구방법론
- Responsible Leadership/Entrepreneurial Mindset/Innovative Vision 관련 통합 모듈

## Repository Goal
이 Git의 목표는 다음 3가지를 한 곳에 체계적으로 남기는 것입니다.

1. 강의 필기/개념 요약 정리
2. AI 활용 흔적(요약, 구조화, 시각자료 보강)
3. 실제 실행 코드/실습 기록

즉, 나중에 빠르게 복습하거나, 해당 과정을 궁금해하는 사람이 전체 학습 흐름을 파악할 수 있도록 만드는 학습 아카이브입니다.

## Writing Style
- 복습 친화형: 핵심 개념 위주로 짧고 구조적으로 정리
- 모듈 분리형: 과목/주제별 폴더로 구분
- 실습 연계형: 설명 + 코드/노트북 + 결과를 함께 기록
- 업데이트형: 학기 진행에 맞춰 지속적으로 보완
