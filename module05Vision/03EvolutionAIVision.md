## Lecture 3: AI Vision의 진화와 인간 시각과의 비교

### 3.1 AI Vision의 발전 단계

#### 진화 과정
```
Perception (지각) → Understanding (이해) → Prediction (예측)
```

#### 모델 발전사
```
AlexNet → ResNet → ViT → V-JEPA
```

#### 패러다임 전환
- **과거**: 데이터 양 증가로 성능 향상
- **현재**: **학습 구조의 혁신**이 AI 성능의 핵심 결정 요인

#### 멀티모달 지능의 등장
```
Vision + Language + Action = Multimodal Intelligence
```

### 3.2 인간 시각 vs 인공지능 시각

#### 인간의 시각 처리
- **의미 기반 해석(Semantic Interpretation)**
  
**처리 과정**:
```
형태 인식 → 맥락 보완 → 기억 참조
```

**뇌 처리 경로**:
```
LGN (외측슬상핵) → V1/V2/V3 (시각피질) → TC (측두피질)
```

#### 인공지능의 시각 처리
- **패턴 기반 통계적 추론(Pattern-based Statistical Inference)**
