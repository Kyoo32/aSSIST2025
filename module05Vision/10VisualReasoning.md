## Lecture 10: Visual Reasoning과 AGI로의 진화

### 10.1 Visual Reasoning: 논리적 추론의 시작

#### 패러다임 전환
```
단순 인식(Recognition) → 논리적 추론(Logical Reasoning)
```

#### Visual Reasoning의 발전 단계
```
Relation (관계 파악)
    ↓
Logic (논리 구성)
    ↓
Causality (인과 관계 이해)
```

### 10.2 Visual Reasoning 처리 파이프라인

```
Object Extraction (객체 추출)
    ↓
Relation Encoding (관계 인코딩)
    ↓
Reasoning Module (추론 모듈)
    ↓
논리적 결론 도출
```

#### 핵심 질문
- "이 객체들은 어떤 관계인가?"
- "왜 이런 배치인가?"
- "다음에 무슨 일이 일어날까?"

### 10.3 CLEVR 벤치마크

#### CLEVR (Compositional Language and Elementary Visual Reasoning)
- **Visual Reasoning 능력의 표준 벤치마크**
- 다양한 형태, 색상, 크기의 객체 관계 추론
- 복합적인 질문-답변 형식

#### 평가 예시
```
질문: "파란 공 왼쪽에 있는 큰 물체는?"
답변: 모델이 객체 간 관계를 이해하고 추론
```
