## Lecture 11 Neural-Symbolic Reasoning

### 11.1 하이브리드 접근법

#### 핵심 개념
**학습 기반 신경망 + 명시적 논리 추론 = Learnable Logic**

#### 구조적 결합
```
신경망 (Neural Network)
- 패턴 학습
- 데이터 기반
- 유연성

    +

기호 논리 (Symbolic Logic)
- 명시적 규칙
- 해석 가능성
- 논리적 일관성

    ↓

Neural-Symbolic AI
- 학습 가능한 논리
- 설명 가능한 추론
```

### 11.2 장점

1. **해석 가능성(Interpretability)**
   - 왜 그런 결론에 도달했는지 설명 가능

2. **논리적 일관성(Logical Consistency)**
   - 모순 없는 추론

3. **데이터 효율성(Data Efficiency)**
   - 논리 규칙으로 적은 데이터로도 학습 가능
