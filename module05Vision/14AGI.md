## Lecture 14 AGI로 가는 길 - 통합 관점

### 14.1 AGI의 두 축

```
Visual Reasoning + Causal Inference = AGI의 핵심
```

#### Visual Reasoning
- **"무엇이 어떻게 연결되어 있는가?"**
- 논리적 추론 능력

#### Causal Inference
- **"무엇이 무엇을 일으키는가?"**
- 인과 관계 이해

### 14.2 통합 AI 시스템의 구조

```
Perception (지각)
    ↓
Understanding (이해)
    ↓
Reasoning (추론)
    ↓
Causal Inference (인과 추론)
    ↓
World Model (세계 모델)
    ↓
Planning & Action (계획 및 행동)
```

### 14.3 현재의 도전 과제

#### 1. 일반화 문제
- 특정 도메인에서 다른 도메인으로의 전이

#### 2. 데이터 효율성
- 인간처럼 적은 경험으로 학습

#### 3. 설명 가능성
- AI의 추론 과정을 인간이 이해 가능하게

#### 4. 인과 관계의 발견
- 관찰만으로 진정한 인과 구조 파악

### 14.4 미래 방향

```
현재: 패턴 인식 + 통계적 추론
    ↓
다음 단계: 논리적 추론 + 인과 추론
    ↓
목표: 일반 인공지능 (AGI)
    ↓
궁극: 인간 수준의 이해와 추론
```
